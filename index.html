SahcharAI – गोपनीयता नीति

अंतिम अद्यतन: 27 मार्च 2026

SahcharAI (“हम”, “हमारा”) आपकी निजता का पूरा सम्मान करता है। यह गोपनीयता नीति बताती है कि हम कौन-सी जानकारी एकत्र करते हैं, उसका उपयोग कैसे करते हैं, और आपके डेटा को सुरक्षित रखने के लिए हम क्या उपाय करते हैं।

1. हम कौन-सी जानकारी एकत्र करते हैं?

हम केवल निम्नलिखित सीमित जानकारी एकत्र करते हैं:

  • चैट संदेश और इमेज प्रॉम्प्ट: जब आप AI चैट का उपयोग करते हैं, तो आपके संदेश DeepSeek API को भेजे जाते हैं। इमेज जनरेशन के लिए आपके द्वारा लिखा गया विवरण (प्रॉम्प्ट) OpenAI DALL-E API को भेजा जाता है। दोनों ही स्थितियों में, यह जानकारी केवल उत्तर या चित्र बनाने के लिए प्रोसेस की जाती है। हम आपके संदेशों या प्रॉम्प्ट को स्थायी रूप से स्टोर नहीं करते।
  • गुमनाम तकनीकी लॉग: ऐप के प्रदर्शन को बेहतर बनाने और क्रैश की समस्या सुलझाने के लिए हम गुमनाम तकनीकी डेटा (जैसे डिवाइस मॉडल, Android वर्जन) एकत्र कर सकते हैं। यह डेटा आपकी व्यक्तिगत पहचान से जुड़ा नहीं होता।
  • फीडबैक फॉर्म से जानकारी: यदि आप हमारे फीडबैक फॉर्म (Google Forms) के माध्यम से सुझाव देते हैं, तो आपके द्वारा दी गई जानकारी (जैसे नाम, ईमेल, रेटिंग) हम तक पहुँचती है। यह जानकारी केवल ऐप को बेहतर बनाने के लिए उपयोग की जाती है।
🔒 हम निम्नलिखित संवेदनशील जानकारी कभी एकत्र नहीं करते:
  • आधार नंबर, पैन कार्ड, या कोई सरकारी पहचान पत्र
  • बैंक खाता, क्रेडिट कार्ड, या भुगतान विवरण
  • पासवर्ड, OTP, या लॉगिन जानकारी
  • लोकेशन, संपर्क सूची, या गैलरी फोटो

2. जानकारी का उपयोग कैसे किया जाता है?

  • चैट और इमेज जनरेशन: आपके सवाल DeepSeek API को, और इमेज प्रॉम्प्ट OpenAI DALL-E API को भेजे जाते हैं। API से उत्तर या चित्र मिलने के बाद वह आपको दिखाया जाता है। हम इस डेटा को लॉग नहीं करते और न ही बाद में उपयोग करते हैं।
  • गुमनाम लॉग: इनका उपयोग केवल ऐप की स्थिरता सुधारने, बग ठीक करने और प्रदर्शन बढ़ाने के लिए किया जाता है।
  • फीडबैक फॉर्म: आपके सुझाव हमें ऐप को अधिक उपयोगी और उपयोगकर्ता-अनुकूल बनाने में मदद करते हैं।

3. तृतीय-पक्ष सेवाएँ और डेटा साझाकरण

  • हम कभी भी आपका डेटा किसी तीसरे पक्ष को नहीं बेचते, न ही मार्केटिंग के लिए साझा करते हैं।
  • DeepSeek API (चैट): हमारा AI चैट फीचर DeepSeek API द्वारा संचालित है। आपके संदेश उनके सर्वर पर प्रोसेस होते हैं। उनकी गोपनीयता नीति: https://platform.deepseek.com/privacy-policy
  • OpenAI DALL-E API (इमेज जनरेशन): इमेज बनाने के लिए आपके द्वारा दिए गए विवरण OpenAI के DALL-E API को भेजे जाते हैं। उनकी गोपनीयता नीति: https://openai.com/policies/privacy-policy
  • Google Forms: फीडबैक लेने के लिए हम Google Forms का उपयोग करते हैं। Google की गोपनीयता नीति उनके प्लेटफॉर्म पर लागू होती है।
